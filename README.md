# 🚀 QuickShare

A real-time file and text sharing web application inspired by Send Anywhere.

## 🔥 Features
- ⚡ Real-time text sharing using Socket.IO
- 📁 File upload and download with unique codes
- 🔑 Room-based communication
- ⏳ Temporary storage with auto-expiry

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Real-time: Socket.IO

## 🌐 Deployment
- Frontend: Netlify
- Backend: Render

## 📌 How it Works
1. User creates a room/code
2. Another user joins using the code
3. Share text instantly or upload files
4. Data expires automatically

## 📷 Future Improvements
- End-to-end encryption
- QR code sharing
- WebRTC peer-to-peer transfer
